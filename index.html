Welcome to uiKit

This basic page is a way (for me) to preview what a bunch of css will render before adding it to a website/app

Usage

add your css files to the root folder (preferably in CSS) then add an item in UIKIT_loader.js following this syntax 


const UI_KITS = [
  {
    name: 'bare',
    longName: 'Bare (no css)', // optionnal
    description: 'no CSS at all',
    css: [
    ]
  },
  {
    name: 'oldproof',
    longName: 'Oldproof (old browsers)', // optionnal
    description: 'using only OOOLD technologies like float for layout',
    css: [
      './Kits/oldproof/layout.css',
      './Kits/oldproof/ui.css'
    ]
  }
]
